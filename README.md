# AI Nexus

AI Nexus is an AI-powered bot management platform integrating OpenRouter AI models with a modern React + Vite frontend and Express backend. It supports user authentication, team collaboration, and chatbot management.

## Features
- User sign up and login with session management
- Bot creation and management with AI-powered responses
- Team workspaces and collaboration tools
- Real-time chat interface with multiple output views
- Designed with TailwindCSS and Radix UI components

## Setup
1. Copy `.env.example` to `.env` and fill in required secrets.
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`

## Folder Structure
- `src/server/` — Backend API, server code
- `src/pages/` — React pages and routes
- `src/components/` — React UI components
- `src/hooks/` — Custom React hooks
- `src/lib/` — Utility functions and query clients

## Scripts
- `npm run dev` — Run both frontend and backend concurrently
- `npm run build` — Build production assets
- `npm run start` — Start production server
